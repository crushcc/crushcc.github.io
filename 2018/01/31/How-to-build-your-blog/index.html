$git init 是在该目录下初始化一个本地的仓库，会在新目录下新建一个.git的隐藏文件夹。
第二条命令是创建一个没有父节点的分支gh-pages，并自动切到该分支
在Git中，分支(branch)的概念非常重要，Git之所以强大，很大程度上就是因为它强大的分支体系。这里的分支名字必须是gh-pages，因为github规定，在项目类型的仓库中，只有该分支中的页面，才会生成网页文件。

遇到过的问题

如果输入

提示出错信息：fatal: remote origin already exists.

解决办法如下： 

1、先输入$ git remote rm origin

2、再输入$ git remote add origin git@github.com:djqiang/gitdemo.git （我的到这儿就解决了）

3、如果输入$ git remote rm origin 还是报错的话，error: Could not remove config section 'remote.origin'. 我们需要修改gitconfig文件的内容

4、找到你的github的安装路径

5、找到一个名为gitconfig的文件，打开它把里面的[remote "origin"]那一行删掉
